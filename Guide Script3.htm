Part 5 : Glossary of Terms

Automation

The application of technology, programs, robotics or processes to achieve outcomes with minimal human input. In this context, it refers to the use of coded instructions that will automate certain tasks on our computer desktop.

 

Macro

In computer programming, a macro is a rule or pattern that specifies how a certain input should be mapped to a replacement output. We use them to automate our inputs to control the desktop.

 

UI.Vision

A software based on Chrome, which can be downloaded and run through Chrome as an extension. It is a tool to create programs called macros, which can control both Chrome and the desktop.

 

Command

In UI.Vision, a command is a specific instruction given in a macro. There are a limited selection of commands, each specifying a different function the line of code will execute.

 

Target

In the context of UI.Vision, a target is the data input which the command uses. It is needed for the line of code to function. It is similar to an argument being passed into a function in conventional programming languages.

 

Value

In UI.Vision, if this field is filled, the output of the line of code(if any) is saved in a variable, the name given by the field. It is not needed for the line of code to function.

 

Description

Works similarly to a comment in conventional coding languages. Data entered into this field has no effect on the macro. However, it makes code easier to understand, especially for others working with your code.

 

Data Scraping

The removal of useful data from a human-readable source. E.g taking the text out of an image.

 

Variables

A variable is an abstract storage location paired with an associated symbolic name, which contains some known or unknown quantity of data or object referred to as a value. In UI.Vision, some variables are built into the software, such as ${KEY_ENTER} for the Enter key

 

File Path

The path from the starting directory to the folder containing the file, combined with the file name and its file extension. For example, C:\Program Files\Windows NT\Accessories\wordpad.exe is the full file path of WordPad.

 

Hotkey

A key or a combination of keys providing quick access to a particular function within your OS or a program youre currently working with. Useful for doing stuff fast with little to no mouse clicks.

 

Screenshot

An image taken by the computer of the entire screen at that moment. In UI.Vision, screenshots are always saved as .png files.

 

.JSON

JavaScript Object Notation, an open standard file format that stores data objects like attribute-value pairs, arrays and other serializable objects. It is commonly used with web-based programs.

 

.HTML

HyperText Markup Language, the standard markup language for web pages to be displayed in a web browser. JavaScript and CSS can be used alongside HTML to make more intractable webpages.

 

.ZIP

An archive file format that supports lossless data compression. Allows you to easily hold, upload and download a compressed and packaged-up amount of folders and files.
